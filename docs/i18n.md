# Internacionalização (i18n)

## Idiomas Suportados

| Código | Idioma | Padrão |
|--------|--------|--------|
| `pt-BR` | Português | ✅ |
| `en-US` | English | |
| `es-ES` | Español | |

## Estrutura

```
/src/i18n/
├── locales/
│   ├── pt-BR.ts
│   ├── en-US.ts
│   ├── es-ES.ts
│   └── index.ts      # Barrel export
└── translations.ts   # Tipos e helpers
```

## Uso

### Hook useTranslation

```tsx
import { useTranslation } from '@/hooks/useTranslation';

function MyComponent() {
  const { t } = useTranslation();
  
  return <h1>{t('app.title')}</h1>;
}
```

### Store i18nStore

```tsx
import { useI18nStore } from '@/stores/i18nStore';

function LanguageSelector() {
  const { locale, setLocale } = useI18nStore();
  
  return (
    <select value={locale} onChange={e => setLocale(e.target.value)}>
      <option value="pt-BR">Português</option>
      <option value="en-US">English</option>
      <option value="es-ES">Español</option>
    </select>
  );
}
```

## Seções de Tradução

| Chave | Conteúdo |
|-------|----------|
| `app` | Título, tagline |
| `timer` | Estados do cronômetro |
| `scramble` | Embaralhamento e guia |
| `stats` | Estatísticas e explicações |
| `advancedStats` | Gráficos e métricas avançadas |
| `sessions` | Gerenciamento de sessões |
| `history` | Tabela de solves |
| `shortcuts` | Atalhos de teclado |
| `settings` | Configurações |
| `onboarding` | Tour interativo |
| `tutorial` | Tutorial layer-by-layer |
| `training` | Modo de treino |
| `navigation` | Links de navegação |
| `actions` | Botões genéricos |
| `penalties` | +2, DNF, nenhuma |

## Adicionando Traduções

1. Adicionar chave em `pt-BR.ts`
2. Adicionar mesma chave em `en-US.ts` e `es-ES.ts`
3. Usar com `t('nova.chave')`

## Persistência

Idioma salvo em `localStorage` via `i18nStore`.
